# curso_analise_de_dados

##Executando o arquivo 'calculadora.sh'
Para executar o arquivo é necessário utilizar o Ubuntu (Linux). 
Utilizar o comando ./calculadora.sh

## Descrição do projeto de Python 'Projeto_1"
O script consiste na determinação do 'número da vida' do usuário, com base na sua data de nascimento.
Sendo assim, o código identifica, via operações matemáticas, o resultado e entrega o significado para o usuário.
